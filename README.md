# CodeClauseInternship_Plagarism_Checker

This is a basic plagiarism checker built using Python. The checker compares two text documents and calculates their similarity using the difflib library. It provides a simple way to identify potential instances of plagiarism.

Features
Compares two text documents for similarity.
Uses the difflib library to calculate similarity ratios.
Detects potential plagiarism based on a predefined threshold.
Requirements
Python 3.x
Libraries: difflib
Installation
Clone or download the repository to your local machine.

No additional libraries are required for installation.

Usage
Ensure that the text documents you want to compare are in the same directory as the script.

Edit the script to set the file paths of the documents you want to compare and adjust the similarity threshold if needed.

Run the script:
python plagiarism_checker.py

The checker will compare the documents and provide a result indicating whether plagiarism is detected or not.
